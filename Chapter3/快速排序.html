第四节 快速排序

快速排序

  • QuickSort – efficient sorting algorithm (visualize)

    • Choose a pivot; move smaller elements left & larger right; sort left & right

  • Best & average case: O(n*log(n)); Worst: O(n2)

  • Memory: O(log(n)) stack space (for recursion)

最坏情况的样子

quick-sort

def quick_sort(arr,lo,hi):
    if lo < hi:
        p = Partion(arr,lo,hi)
        quick_sort(arr,lo,p)
        quick_sort(arr,p+1,hi)

Lomuto partioning

从左到右找分割点

选择pivot,放在最左边 定义store索引来存储即将下一个比pivot小的数 从左到右进行遍历，直到结束 定位到pivot应该出于的位置，此时是store(表示存储下一个比pivot小的数)应该减1，store-1的位置就是pivot的位置 交换最左边0与store-1的元素， 则store-1就是分割点

# Lomuto partioning
def Partion(arr,lo,hi):
    # 枢纽点
    pivot = arr[lo]
    store = lo+1
    for i in range(store,hi+1):
        if arr[i] <= pivot:
            swap(arr,i,store)
            store += 1
    store -= 1
    swap(arr,lo,store)
    return store

Hoare Partitioning

两头并进找分割点

初始化
left = lo+1
right = hi

循环：
    当arr[left] < pivot时，left+1前进，直到hi
    当arr[right]>pivot时，right-1前进，直到right减到lo时，此时的arr[right] = arr[lo]
    如果right< left 退出，
    否则 swap(right,left)继续循环

交换pivot与right的元素
返回rights

# Hoare Partitioning
def Hoare_Partion(arr,lo,hi):
    pivot = arr[lo]
    left = lo+1
    right = hi
    while True:
        while left<=hi and arr[left] <= pivot:
            left += 1
        while arr[right] > pivot:
            right -= 1
        if right<left:
            break
        else:
            swap(arr,left,right)
    swap(arr,lo,right)
    return right

def swap(arr,i,j):
    arr[i],arr[j] = arr[j],arr[i]
